# 自主决策记录

> 用户授权：实现过程中不确定的点先按我的判断决定，统一记录，完成后汇报。
> 已确认的大方向：纯字形美术风格；不做提示系统（题面即提示）；游戏名《你从来没玩过的完形填空》；关卡名在未通关前不外露谜底。

## 全局

- **G1 技术栈**：零构建（HTML+CSS+原生 ES 模块），用 `python -m http.server` 跑。原因：本机 npm 损坏、PowerShell 无 node，只有 Python 通用。
- **G2 关卡名不外露谜底**：选关卡片未通关时只显示挖空题面（如「＿能载舟」），通关后才显示完整成语。题面里的填空用红色下划线，已填用绿色。
- **G3 每个有结果的动作都配特效**：建了通用特效库 `effects.js`（噗/闪光/烟/水滴/水波/火星/飘字/抖动/弹跳）与复用机制 `mechanics.js`。同类交互跨关复用（火烤、融合、斧砍、旋转、翻转等都是同一套代码与动效）。
- **G4 解锁方式**：默认线性解锁逻辑已写好，但**开发/验收期所有关卡都可直接点开**（方便你逐关验收），不强制按顺序。上线时可一行开关改回强制线性。
- **G5 无音效/背景音乐**：本期先不做，留待后续。
- **G6 多字 token**：词素材（铁杵/赵高/种子/树桩/指认）用更宽、字号略小的 token 呈现。

## 第一章

- **L2 旋转手势（正→歪）**：按住字横向拖动即倾斜，转过约 42° 松手即“扳歪”，定格一个 16° 倾角并变成「歪」，随后可拖入填空。动作（扳斜）即语义。
- **L3 翻转手势（下→上）**：按住字向上掀起约 60px 松手即翻面，翻面瞬间换成「上」。
- **L4 融合判定**：把一个字拖到另一个字上、中心距 < 74px 即按规则合成；合成时被拖的字吸附到目标上、闪光+噗，目标变成结果字。两步：口+口→日、日+月→明。

## 第二章

- **L5 排列成河**：给 6 个「水」，中央有一条虚线“河道”。把「水」拖进河道会吸附排列并锁定；放满 4 个即汇流合成「河」。（“很多水”量化为放满 4 个触发。）
- **L6 斧砍**：把「斧」拖到树上即砍一下，森→林→木 逐级递减，每砍飞出一片“木”碎片。复用通用斧砍机制。
- **L7 网捞鱼 + 鱼羊成鲜**：底部是水池，把「网」拖进水池区域即捞出一条「鱼」（一次）；再把「鱼」拖到「羊」上融合成「鲜」。
- **L8 种子生长**：「水」「日(阳光)」是**可重复使用的工具**，用完自动归位。给种子先浇水+晒太阳→发芽成「苗」；再给苗浇一次水→长成「木」。
- **L9 炎与淡**：复用融合机制。火+火→炎（火光特效）；水+炎→淡（蒸汽浇灭特效，并去掉火色改水色）。

## 第三章

- **L10 吹草显物（双填空）**：把「风」拖到「草」旁（距离 < 98px）即触发，草左右摇摆，从两侧冒出「牛」「羊」，分别拖进两个空。两个空分别只接受对应的字。
- **L11 三人成虎**：把三个「人」聚到一起（两两距离都 < 122px）即触发：每个人头顶冒“有虎！”，随后三人消散、凭空生出一只「虎」。
- **L12 指鹿为马**：复用融合。赵高+指→「指认」（权势红色词 token）；指认+鹿→马（红色权势特效 + 飘“指鹿为马！”）。赵高、指认用词 token 呈现。
- **L13 杯弓蛇影**：复用邻近蓄能（同火烤冰）。把「弓」举到酒杯旁停住约 1.4s，杯面荡起水波、浮出「蛇」。酒杯是不可拖的布景 token。
- **L14 雪中送炭（三步）**：复用斧砍（森→林→木）+ 邻近加热（火烤木→炭）。火只有在砍到「木」之后才点得着（enabled 判定 value==='木'），炭化时冒烟。

## 第四章

- **L15 铁杵磨成针（持续摩擦）**：把「磨」按住在「铁杵」上来回蹭，统计左右换向次数，需 **10 个来回**；每蹭一下迸火星、铁杵横向变细（scaleX 收缩），磨满变「针」。
- **L16 守株待兔（等待）**：把「树桩」放下即开始等待（即便完全不动，进入 3s 后也会自动开始），约 3.2s 后一只「兔」从右侧飞奔撞上树桩，撞击后变为可拖动的「兔」。**用真实时间的“等”作为解法**。
- **L17 釜底抽薪（横向思维 + 诱饵）**：灶上有釜、釜下有火、火下压着「薪」，旁边一杯「水」是诱饵。往火上浇水 → 飘“杯水车薪…”、水自己弹回原位、火不灭；把「薪」从灶下抽离（离灶台中心 > 118px）→ 火立刻灭、飘“釜底抽薪！”，随后「薪」可填入。
- **L18 画蛇添足（克制陷阱）**：直接把「蛇」拖进空即过关。若把「足」拖到「蛇」上 → 蛇长腿、抖动、向右窜出屏幕逃走，整关重置（重新生成蛇与足），需重来。
